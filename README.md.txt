# Proyecto SST - SENA

Proyecto académico desarrollado para el SENA.

## Estructura del proyecto

- frontend: HTML, CSS y JavaScript
- backend: lógica Java
- proyecto-sena-hibernate: Hibernate + MySQL
- proyecto-sena-web: Servlets y JSP

## Tecnologías usadas
- Java
- Maven
- Hibernate
- MySQL
- HTML, CSS, JavaScript
- Git y GitHub
